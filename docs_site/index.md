# UniAmp Overview

UniAmp (Unique Amplicon) is a computational pipeline to generate PCR primers specific to a target genome.<br>
<br>
The UniAmp pipeline can be conceptually split into 4 parts:<br>
1. Build directory of query genomes with high sequence similarity to target genome.<br>
2. Retrieve unique sequences in a target genome compared to query genomes.<br>
3. Select unique target sequence for primer design.<br>
4. Design primers to unique target sequence.<br>
